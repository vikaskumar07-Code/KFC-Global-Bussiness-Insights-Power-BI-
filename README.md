# KFC Global Bussiness Insights (Power BI)

## Project Overview
This project presents an interactive **Power BI dashboard** analyzing global sales performance of KFC across multiple countries and branches. The dashboard provides insights into sales trends, customer distribution, marketing efficiency, and branch-level performance.

## Problem Statement
KFC operates across multiple countries and branches, but business performance is not uniform across all locations.
The objective of this project is to:
* Identify **top-performing and underperforming regions and branches**
* Analyze the relationship between **sales, customers, and marketing spend**
* Evaluate **marketing efficiency and profitability**
* Provide actionable insights to improve overall business performance

## Dashboard Pages

### 1. Overview Page

* Displays key KPIs:
  * Total Sales
  * Total Customers
  * Marketing Spend
  * Profit
* Provides a high-level summary of global performance

### 2. Country Analysis

* Sales comparison across countries
* Customer distribution (Pie Chart)
* Marketing Spend vs Sales trend

#### Key Insights:

* Canada and UK contribute the highest sales
* USA shows the lowest performance
* Marketing spend does not always lead to higher sales

### 3. Branch Performance

* Top 10 branches by customer count
* Top 5 branches by sales & profit
* Detailed branch-level table (Sales, Marketing, Profit, Customers)

#### Key Insights:

* Top-performing branches are concentrated in Canada and UK
* Strong correlation between customer count and sales
* Some branches achieve higher profit with optimized marketing spend
* Significant performance gap between top and low-performing branches

## Key Metrics Used

* **Total Sales**
* **Total Customers**
* **Marketing Spend**
* **Profit (Sales - Marketing Spend)**
* **Average Order Value (AOV)**
* **Marketing Efficiency**

## Tools & Technologies

* Power BI
* DAX (Data Analysis Expressions)
* Data Modeling
* Data Visualization

## 📁 Dataset
FROM Kaggle 
* Dataset Name: `KFC_Past_Sales_Dataset` 
* Includes:
  * Country
  * Branch Name / Branch ID
  * Sales
  * Customers
  * Marketing Spend
  * Profit
  * Date (Year / Month)

## Key Learnings

* Created interactive dashboards using Power BI
* Applied DAX for calculated measures and KPIs
* Used filters, Top N, and conditional formatting
* Designed business-focused data storytelling dashboards

## Conclusion

This dashboard helps stakeholders identify high-performing regions and branches while highlighting areas needing improvement. By optimizing marketing strategies and replicating successful branch models, overall business performance can be enhanced.
